# TON Storage部署包

这个文件夹包含了部署到TON Storage所需的文件：
- `index.html` - 您的网站主页
- `favicon.ico` - 网站图标

## 部署到TON Byte
1. 将这整个文件夹上传到 [tonbyte.com](https://tonbyte.com)
2. 获得Bag ID
3. 在 dns.ton.org 的"TON Storage"字段填入Bag ID
4. 完成！您的 greatestofalltime.ton 将通过完全去中心化的方式运行

## 或者手动部署
如果您熟悉TON工具，可以使用TON Storage CLI工具直接部署这个文件夹。
